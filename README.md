# DentalGeom

A 3D dental mesh viewer and crown restoration generator built with Three.js and TypeScript.

## Features
- STL file loading and visualization
- Interactive 3D controls
- Mesh analysis and statistics
- Crown geometry generation (coming soon)

## Tech Stack
- Three.js / WebGL
- TypeScript
- Vite

## Getting Started
bash <br />
npm install <br />
npm run dev <br />
